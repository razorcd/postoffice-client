API Inspector

An application that will assist you in debugging your http API requests.

It is very easy to use. Just redirect your requests to the following url: "localhost:8080/incoming" and you will see a full description of all your request data and metadata.

Feel free to add any aditional path after this URL. Also you can use any request method (GET, POST, PUT, ...) add any params, headers, cookies and body data.

Soon we will also support uploading files.

All the successful requests will get a response status 204 No Content.
